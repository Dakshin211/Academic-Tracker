# Student Management System

A simple desktop-based student management application developed using Python, Tkinter, and MySQL.

## Features
- Add, update, and delete student records
- View student details
- MySQL database integration
- Simple Tkinter graphical interface

## Tech Stack
- Python
- Tkinter
- MySQL

## Setup

```bash
git clone https://github.com/yourusername/student-management-system.git
cd student-management-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Configure your MySQL database and run:

```bash
python main.py
```

## Project
This project was created to simplify basic student record management using a desktop application.
